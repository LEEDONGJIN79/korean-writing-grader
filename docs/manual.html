🤔 시작하기 전에 — 이 도구가 뭔가요?

Korean Writing Grader는 국어 선생님이 학생들의 서·논술형 글(시 감상문, 논설문, 자기소개서 등)을 채점할 때 여러 명의 AI가 토론해서 도와주는 도구입니다.

이런 분께 도움이 됩니다

  • 학기말에 30명 학급의 글을 채점하느라 매일 야근하시는 선생님
  • 혼자 채점하다 보니 학생마다 일관되게 평가했는지 걱정되시는 선생님
  • 학생에게 줄 피드백을 더 구체적이고 풍부하게 쓰고 싶은 선생님
  • AI를 교실에 어떻게 활용할지 고민하시는 선생님

이 도구가 해주는 것

  1. 학생 글을 실제 학교 평가 절차(채점교사→학생이의→검토교사→의장 4단계)로 채점
  2. 의견이 갈리면 외부 AI(ChatGPT, Gemini)에게 한 번 더 확인
  3. 그래도 애매하면 "선생님 확인 필요"로 표시
  4. 학생용 리포트, 교사용 검토 자료, 채점표(엑셀)를 자동으로 만들어 줌
⚠️ 한 가지만 미리 알려드릴게요:
최종 점수는 항상 선생님이 확정합니다. AI는 보조 도구이지, 선생님을 대체하지 않습니다.

필요한 시간

작업예상 시간
설치 (이 안내서 따라하기)약 30분 (처음 한 번만)
한 학급 채점 (30명 기준)약 2~3시간 (선생님이 직접 하면 5~7시간)

한 번 설치하면 이후로는 평가할 때마다 30분 안에 끝납니다. 자, 시작합시다.

1️⃣ 1단계: Claude Desktop 설치하기

이 도구는 Claude Desktop이라는 프로그램 위에서 작동합니다. Claude Desktop은 AI 회사 Anthropic에서 만든 무료 프로그램이에요.

1웹 브라우저(크롬, 사파리, 엣지 등)를 열어주세요.
2주소창에 다음을 입력하고 엔터: 
https://claude.ai/download
3자기 컴퓨터 종류에 맞는 버튼을 찾아 클릭하세요.
  • 맥(MacBook, iMac 등) 사용자 → "Download for Mac"
  • 윈도우 PC 사용자 → "Download for Windows"
4다운로드된 파일을 더블클릭해서 실행하세요.

맥에서는 Claude.dmg, 윈도우에서는 Claude-Setup.exe 같은 파일이 다운로드됩니다.

🍎 맥 설치 방법 (더 자세히)

  1. Claude.dmg 파일을 더블클릭하면 작은 창이 하나 뜹니다.
  2. 창 안에 Claude 아이콘과 "Applications" 폴더 아이콘이 보입니다.
  3. Claude 아이콘을 마우스로 끌어다(드래그) Applications 폴더 위에 놓으세요.
  4. "복사 중..." 진행 표시가 끝나면 설치 완료.
  5. Launchpad 또는 Spotlight(⌘ + 스페이스바)에서 "Claude"를 검색해 실행.

🪟 윈도우 설치 방법 (더 자세히)

  1. Claude-Setup.exe 파일을 더블클릭.
  2. "이 앱이 변경을 수행하도록 허용하시겠습니까?" 창이 뜨면 예 클릭.
  3. 설치 마법사가 자동으로 진행됩니다. 다음, 다음 누르세요.
  4. 설치 완료 후 시작 메뉴에서 "Claude"를 찾아 실행.
💡 팁: 처음 실행하면 로그인 화면이 나타납니다. 다음 단계(2단계)에서 계정을 만든 후 다시 돌아오세요.

✅ 1단계 완료 확인

  • 내 컴퓨터에 Claude Desktop 프로그램이 설치되어 있다.
  • Claude 아이콘을 더블클릭하면 프로그램이 열린다.

2️⃣ 2단계: Claude 구독하기

Claude는 기본적으로 무료지만, 무료 플랜은 하루에 사용할 수 있는 횟수가 제한되어 있어 학급 채점에 부족합니다. Pro 플랜($20/월, 약 27,000원)을 구독하면 학급 전체 채점이 가능합니다.

💰 비용 안내
· Pro: 월 $20 (약 27,000원) — 학급 1~2개 채점에 충분
· Max ($100~200/월): 학교 전체나 자주 사용 시
· 결제는 신용카드만 받습니다.
1웹 브라우저에서 https://claude.ai 접속
2"Continue with Google" 또는 "Sign up with email" 중 선택해 가입

3가입 후 좌측 상단의 메뉴(≡)를 클릭하고 "Settings" 또는 "설정"
4"Plan" 또는 "요금제" 메뉴 클릭
5"Upgrade to Pro" 또는 "Pro로 업그레이드" 클릭
6카드 정보 입력 후 결제

한국 카드도 대부분 결제 가능합니다.

⚠️ 학교 카드/공금으로 결제할 때 주의:
학교 회계 규정에 따라 사전 승인이 필요할 수 있습니다. 결제 전 행정실 또는 교감 선생님께 확인해 주세요.
7구독 완료 후 Claude Desktop 프로그램을 다시 실행해 로그인

✅ 2단계 완료 확인

  • Claude Desktop에 로그인되어 있다.
  • 좌측 상단에 내 계정 이름이 보인다.
  • "Pro" 또는 "Max" 표시가 보인다.

3️⃣ 3단계: 파일 권한 켜기 (가장 중요)

Claude가 선생님 컴퓨터의 파일을 읽고 채점하려면 파일 접근 권한이 켜져 있어야 합니다. 이걸 켜지 않으면 4단계부터 안 됩니다.

⚠️ 학교 컴퓨터에서는 보안 정책 때문에 안 켜질 수 있습니다.
개인 노트북이나 집 컴퓨터에서 사용하시거나, 학교 정보 부서에 문의하세요.

🍎 맥에서 권한 켜는 방법

1Claude Desktop 실행
2화면 상단 메뉴바에서 "Claude" → "Settings..." (또는 ⌘ + ,) 클릭
3"Developer" 또는 "개발자" 탭 찾기
4"Local Agent Mode" 또는 "Allow Claude to access local files" 항목을 켜기 (ON)
5처음 켤 때 macOS가 권한 요청 팝업을 띄울 수 있습니다 → "허용" 클릭

🪟 윈도우에서 권한 켜는 방법

Claude Pro 구독료만 듭니다 (월 약 27,000원). ChatGPT·Gemini는 무료 계정으로도 외부 검증 가능합니다. 한 학기에 평가 5번 한다면 한 평가당 5,400원 정도입니다.

Q2. 학생 자료가 외부로 새지 않나요?

아니요. 학생 자료는 선생님 컴퓨터 안에서만 처리됩니다. 외부 모델 검증 시에도 학생 이름·학번은 자동으로 익명화됩니다. GitHub에도 절대 업로드되지 않도록 차단되어 있습니다.

Q3. 한글(.hwp) 파일이 안 읽혀요

한글 파일은 변환이 필요합니다. 한글 프로그램에서 "파일 → 다른 이름으로 저장 → Word(.docx) 또는 PDF"로 변환해 주세요. 윈도우에 한글이 설치되어 있으면 일부 변환 도구를 통해 자동 처리도 가능합니다.

Q4. 시 감상문 말고 다른 평가에도 쓸 수 있나요?

네. 어떤 서·논술형 평가에도 사용 가능합니다. 시 창작, 논설문, 감상문, 자기소개서, 인터뷰 정리문, 보고서 등. 루브릭만 그 평가에 맞게 바꾸면 됩니다.

Q5. AI가 매긴 점수를 학생에게 그대로 줘도 되나요?

아니요. 반드시 선생님이 검토 후 확정한 점수를 줘야 합니다. AI 점수는 보조 자료입니다. 특히 🔴 필수검토로 표시된 학생은 직접 검토하지 않으면 안 됩니다.

Q6. 학생이 이의제기를 하면 어떻게 하나요?

속기록.md 파일이 가장 강력한 증거 자료입니다. AI 6명의 판정 과정과 외부 모델 검증 결과가 모두 기록되어 있어, 채점의 근거를 명확히 보여줄 수 있습니다.

Q7. Claude가 채점을 잘 못 하는 것 같아요

몇 가지 점검해 보세요:

  • 루브릭이 명확한가? (예: "충분히" 같은 모호한 표현 줄이기)
  • 함정 패턴 메모를 빠뜨렸나? (영역별로 흔한 오해 미리 알려주기)
  • 회색지대 학생을 외부 검증으로 풀었나?
  • 그래도 안 되면 그 학생은 직접 채점하시는 게 정답입니다.

Q8. 학교 컴퓨터에서도 쓸 수 있나요?

학교 정보 부서의 보안 정책에 따라 다릅니다. Local Agent Mode가 막혀 있으면 안 됩니다. 개인 노트북이나 집 컴퓨터에서 사용하는 게 일반적입니다.

Q9. 한 번 설치한 후 새 평가를 채점하려면?

Claude Desktop을 열고 채팅창에 "새 평가 시작할게..."라고 시작하면 됩니다. 다시 설치할 필요 없어요. 5단계(평가 준비)부터 시작합니다.

Q10. 동료 교사와 함께 사용하고 싶어요

각자 자기 컴퓨터에 설치해 사용하면 됩니다. 학생 자료는 각자 분리되어 있어 서로 안 보입니다. 다만 루브릭·과제정보는 공유 가능합니다. (이메일로 보내거나 GitHub 저장소를 함께 사용)

🆘 막혔을 때 도움받기

먼저 확인할 것

  1. Claude Desktop이 최신 버전인지 (메뉴에서 "Check for Updates")
  2. Local Agent Mode가 켜져 있는지 (3단계 다시 확인)
  3. 인터넷 연결이 안정적인지
  4. Claude Pro 구독이 유효한지

그래도 안 되면

  • 어느 단계에서 막혔는지 (이 안내서의 단계 번호)
