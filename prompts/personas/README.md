# 페르소나 라이브러리 — v0.9

v0.9부터는 **실제 학교 평가 절차를 모방한 4단계 페르소나 모델**입니다.

## 페르소나 8개 구성

```
[학생 글]
   ↓
① 채점 교사 (1명) — 사용자가 선택
   · 채점교사_엄격.md
   · 채점교사_느슨.md
   ↓
② 학생 (모든 학생) — 1종 고정
   · 학생.md
   ↓ (학생 이의 있을 때만)
③ 검토 교사 (2명 병렬) — 2종 고정
   · 검토교사_보수.md
   · 검토교사_탐색.md
   ↓
④ 의장 (1명) — 사용자가 선택
   · 의장_엄격.md
   · 의장_균형.md
   · 의장_느슨.md
```

## 사용자가 시작 시 선택하는 것

채점 시작 전 Claude가 묻기:
1. **채점 교사 수준**: 엄격 / 느슨
2. **의장 수준**: 엄격 / 균형 / 느슨

기본 권장 조합: **채점교사 엄격 + 의장 균형**

## 호출 흐름

| 단계 | 페르소나 | 빈도 |
|:---:|---|:---:|
| 1 | 채점 교사 (선택된 1종) | 학생당 1회 |
| 2 | 학생 | 학생당 1회 (전원) |
| 3 | 검토교사 보수·탐색 (2명 병렬) | 이의 있을 때만 |
| 4 | 의장 (선택된 1종) | 학생당 1회 |

학생당 평균 약 3.7 호출 (이의 있는 학생만 5회).

## v0.2~v0.8 페르소나는 어디로?

기존 6개 평가 철학 페르소나(형식주의자·시도_옹호자·텍스트_충실파·맥락_통합파·의심파·호의파)는 `_archive_v02/` 에 보존되어 있습니다. 참고용으로만 유지되며, v0.9는 사용하지 않습니다.

v0.2~v0.8이 "평가 철학 4~6명 병렬 토론" 모델이었다면, **v0.9는 "실제 학교 이의제기 절차" 모델**입니다. 더 자연스럽고 학생 시각이 포함됨.
