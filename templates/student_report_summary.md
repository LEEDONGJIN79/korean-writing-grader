# 학생 리포트 — 권장검토용 (요약)

🟡 **권장검토** 학생용 양식. 회색지대 1건 또는 확신도 평균 60~80%.

전체 분량 **500~700자**. 풀 리포트(2000~3000자)의 약 1/4 수준.

## 핵심 차이점 (풀 리포트와 비교)

| 항목 | 풀 리포트 | 요약 리포트 |
|---|---|---|
| 전체 분량 | 2000~3000자 | 500~700자 |
| 항목별 결과 | 항목마다 풀 설명 + 인용 | 한 줄씩 |
| 강점 | 2~3개, 인용 풍부 | **1개만**, 인용 1개 |
| 보완점 | 2~3개, 자세히 | **1개만** |
| 성취기준 도달 | 표 + 한 문단 | 한 문장 |
| 선생님의 한 마디 | 별도 단락 | 회색지대 안내로 대체 |

## 요약 시 무엇을 줄이는가

- ❌ 풀 리포트의 "선생님의 한 마디" 통합 코멘트 (대신 강점·보완에 흡수)
- ❌ 항목별 풀 설명 → 한 줄 사유
- ❌ 강점 여러 개 → 핵심 1개만
- ❌ 보완점 여러 개 → 핵심 1개만
- ❌ 성취기준 도달 표 → 한 문장
- ✅ 학생 글 인용은 유지 (최소 1개)
- ✅ 회색지대 안내는 유지 (학생이 점수 사유 알 권리)

## 속기록(요약 버전)도 함께

권장검토 학생은 속기록도 요약 버전으로 생성:
- 풀 속기록(transcript_raw.md)은 그대로 보관
- 학생용 속기록(속기록_요약.md)은 핵심 발언 3~5개만 발췌

자세한 내용은 `templates/debate_transcript.md` 참조.
