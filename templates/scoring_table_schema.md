# 채점표 엑셀 스키마

채점표 엑셀은 `anthropic-skills:xlsx` 스킬로 동적으로 생성됩니다. 이 파일은 엑셀의 구조·시트·서식 규칙을 명세합니다.

---

## 파일명
`{{과제명}}_채점표_{{날짜}}.xlsx`

예: `시창작_해설서_채점표_2026-05-24.xlsx`

---

## 시트 구성 (5개)

### 📋 시트 1: 채점표 (메인)

**컬럼 구조**:

| 컬럼 | 설명 | 형식 |
|---|---|---|
| A | 학번 | 텍스트 |
| B | 이름 | 텍스트 |
| C ~ (N+2) | 조건①, 조건②, ... 조건N | 텍스트 + 확신도 |
| N+3 | 잠정 점수 | 숫자 |
| N+4 | 상한 점수 | 숫자 |
| N+5 | 회색지대 개수 | 숫자 |
| N+6 | 검토 라벨 | 🟢/🟡/🔴 |
| N+7 | 교사 확정 점수 | 빈 칸 (교사가 입력) |

**셀 표기 규칙 (조건 컬럼)**:
- 충족 + 확신도 80%↑: `O (95%)`
- 충족 + 확신도 60~80%: `O (70%)`
- 회색지대: `? (50%)`
- 미충족 + 확신도 60%↑: `X (85%)`

**조건부 서식**:
- 확신도 < 60%: 배경 노란색
- 회색지대 `?`: 배경 빨간색
- 검토 라벨 🔴: 행 전체 옅은 빨강
- 검토 라벨 🟡: 행 전체 옅은 노랑

**셀 메모 (마우스 올리면 표시)**:
- 각 조건 셀: 평가자 `원_발언` 100자 요약 + 핵심 인용

## 서식 규칙 종합

### 폰트
- 헤더: 굵게, 12pt
- 본문: 일반, 10pt
- 한글 폰트: 맑은 고딕 또는 시스템 기본

### 색상 코드
- 노랑 (60% 미만 확신도): `#FFF3CD`
- 빨강 (회색지대): `#F8D7DA`
- 옅은 빨강 (필수검토 행): `#FFEBEE`
- 옅은 노랑 (권장검토 행): `#FFFDE7`
- 옅은 초록 (자동확정 행, 선택): `#E8F5E9`

### 행 고정
- 첫 행 (헤더): 항상 보이도록 고정
- 첫 컬럼 (학번): 가로 스크롤 시 고정

### 셀 너비
- 학번·이름: 자동 맞춤
- 조건 컬럼: 100px 정도
- 코멘트 컬럼: 300~500px (시트 2에서)

---

## 생성 시 Claude의 처리

1. 모든 학생 처리가 끝났는지 확인
2. `anthropic-skills:xlsx` 스킬 호출 (시트 5개, 위 명세대로)
3. 데이터를 의장 보고(`03_chairman.yaml`)에서 가져와 채움
4. 조건부 서식·셀 메모·고정 적용
5. `outputs/{{평가명}}/채점표.xlsx`로 저장
6. 사용자에게 검토 우선순위(🔴 학생 명단)와 함께 안내

## CSV 대체 출력 (엑셀 생성 실패 시)

- `채점표.csv` — 시트 1만 CSV로
- `개인별코멘트.csv` — 시트 2만 CSV로
- `검토우선순위.csv` — 시트 5만 CSV로
